# Daily Task Manager

Daily Task Manager is a simple web application that helps users manage their daily tasks easily.

## Features
- Add new tasks
- Display task with date and time
- Mark task as completed
- Delete tasks individually
- Simple and colorful user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
daily-task-manager/
│
├── index.html
├── style.css
├── script.js
└── README.md


## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`
4. Start adding your daily tasks
